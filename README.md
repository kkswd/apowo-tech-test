# 3D 沉浸式年会抽奖系统

## 🚀 如何体验项目

本项目是纯前端零依赖架构，无需配置 Node.js 环境。可通过以下方式快速体验：

**本地运行测试：**
1. 点击本页面右上角绿色的 `<> Code` 按钮，选择 **`Download ZIP`** 将项目下载到本地。
2. 解压文件夹后，直接双击打开 **`index-v6-explode.html`** 文件（这是最终包含所有闭环与特效的主文件）。
3. 推荐使用 Google Chrome 或 Edge 浏览器打开。

> 📂 **关于文件结构的说明：**
> 仓库内保留了 `index-v2` 到 `v6` 的多个历史版本文件。这是为了在《AI 编程报告》中更好地展示我与 AI 进行**渐进式迭代**（从基础动效到 3D 重构，再到业务逻辑闭环）的完整工作流。

## 🛠️ 核心功能亮点
- **极简底层架构**：拒绝臃肿，不依赖 Three.js 等重型 3D 库，纯基于 CSS3 原生硬件加速实现 3D 标签云球体。
- **业务数据闭环**：包含左侧实时奖项库存管理，以及右侧抽屉式中奖名单沉淀看板。
- **震撼视觉动效**：基于 CSS `cubic-bezier` 的物理回弹效果与 `canvas-confetti` 全屏彩带喷射。
