# 3D 沉浸式年会抽奖系统

## 🚀 项目启动说明（极其简单）
本项目采用纯原生 HTML + CSS3 + Vanilla JavaScript 开发，**零依赖，无需安装任何 Node.js 环境或构建工具**。

1. 解压本文件夹。
2. 请使用现代浏览器（强烈推荐 **Google Chrome** 或 Microsoft Edge）打开。
3. 直接双击打开本目录下的 `index.html` 文件，即可直接体验。

## 🛠️ 核心功能亮点
- **极简底层架构**：拒绝臃肿，不依赖 Three.js 等重型 3D 库，纯基于 CSS3 原生硬件加速实现 3D 标签云球体。
- **业务数据闭环**：包含左侧实时奖项库存管理，以及右侧抽屉式中奖名单沉淀看板。
- **震撼视觉动效**：基于 CSS `cubic-bezier` 的物理回弹效果与 `canvas-confetti` 全屏彩带喷射。
